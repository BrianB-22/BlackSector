# 1. Purpose

The Countermeasures System provides defensive capabilities against incoming attacks.

This system is critical for large ships that cannot rely on stealth.

---

# 2. Types of Countermeasures

Examples include:

* electronic warfare
* decoy drones
* missile interception
* sensor jamming

---

# 3. Core Mechanics

Countermeasure evaluation occurs during incoming attack resolution.

Sequence:

```
incoming attack detected
→ countermeasure activation
→ interception probability check
→ attack continues or neutralized
```

---

# 4. Balancing Principles

Countermeasures should:

* reduce incoming damage
* create tactical decision points
* not guarantee survival
